# python-utils

Uma coleção de utilitários em Python feitos pra automatizar tarefas úteis, aprender programação na prática e criar ferramentas realmente legais no dia a dia.

A ideia desse repositório é simples:

> criar mini projetos úteis, gratuitos e fáceis de modificar.

Aqui você vai encontrar desde automações simples até ferramentas com IA, visão computacional, organização de arquivos, manipulação de imagem e aplicações web.

## Alguns exemplos do que vai ter aqui

* Mini ChatGPT local com Flask + Ollama
* Conversor em massa de imagens
* Organizador inteligente de arquivos
* Editor automático de fotos
* Resumidor de PDFs com IA
* Transcritor de áudio/vídeo
* Ferramentas para produtividade
* Sistemas estilo RPG/gamificação
* Utilitários para jogos e desenvolvimento
* Scripts aleatórios úteis que surgirem no caminho

## Objetivo

Esse projeto existe pra:

* aprender Python construindo coisas reais
* compartilhar ferramentas úteis gratuitamente
* estudar backend, automação e IA
* servir como laboratório de ideias

## Tecnologias usadas

Algumas tecnologias que podem aparecer nos projetos:

* Python
* Flask
* Ollama
* OpenCV
* Pillow
* Whisper
* HTML/CSS/JS
* APIs
* Automação de arquivos

## Estrutura

Cada utilitário terá sua própria pasta, documentação e instruções de uso.

Exemplo:

```bash
python-utils/
│
├── mini-gpt/
├── image-converter/
├── pdf-summarizer/
├── auto-organizer/
└── ...
```
